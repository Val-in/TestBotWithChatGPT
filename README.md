# Telegram ChatGPT Bot

Телеграм бот на Python, который отвечает на запросы через API ChatGPT.

## Установка

1. Клонируйте репозиторий:

`git clone <your-repository-url>
cd <repository-folder>`

2. Создайте виртуальное окружение и активируйте его:

`python -m venv .venv` <br>
`.venv\Scripts\activate`

3. Установите зависимости:

`pip install -r Requirements.txt`

4. Создайте файл .env с ключами:

`TELEGRAM_TOKEN=<ваш токен>` <br>
`OPENAI_API_KEY=<ваш ключ OpenAI>`

5. Запустите проект.

Бот будет отвечать на команды /start и /help, а также на любые текстовые сообщения.
